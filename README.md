# HTML5 &amp; CSS3

Este repositório contém os estudos sobre **HTML5 e CSS3** através do [Curso em Vídeo](https://cursoemvideo.com) e [Hcode](https://hcode.com.br)

Existem 3 pastas principais:
- [Exercícios](https://github.com/nadsondejesusalmeida/html-css/tree/main/exercises): Contém os principais exercícios feitos através do **Curso em Vídeo**.
- [Desafios](https://github.com/nadsondejesusalmeida/html-css/tree/main/challenges): Contém os desafios propostos pelo **Curso em Vídeo**.
- [Bônus](https://github.com/nadsondejesusalmeida/html-css/tree/main/bonus): Estudos extras que foram estudados através da **Hcode**.
