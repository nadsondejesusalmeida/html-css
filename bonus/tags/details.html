Details

A tag <details> em HTML é usada para criar um widget interativo que permite ao usuário exibir ou ocultar informações adicionais.

<summary> é usado dentro de <details> para fornecer um título ou cabeçalho para a seção recolhível. Este título é sempre visível, mesmo quando o conteúdo está oculto.
